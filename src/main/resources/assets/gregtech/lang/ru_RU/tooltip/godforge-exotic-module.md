Это модуль Годфорджа
Должен быть частью Годфорджа для работы
Используется для дегенерации материи при сверхвысоких температурах
{dark_gray:{hr}}
Четвёртый и последний модуль Годфорджа, этот модуль расщепляет фундаментальные строительные блоки материи, производя в процессе экзотические смеси. Кварк-глюонная плазма может быть получена сразу же, но для производства магнитной монопольной материи (магматерии) требуется полностью улучшенный Годфордж
Этот модуль специализируется на получении уникальных материалов
