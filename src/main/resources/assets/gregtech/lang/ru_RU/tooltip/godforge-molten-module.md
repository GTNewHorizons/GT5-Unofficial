Это модуль Кузницы Богов
Для работы должен быть частью Кузницы Богов
Используется для высокотемпературного сжижения материалов
Это второй модуль Кузницы Богов, который плавит материалы непосредственно в жидкую форму. Если материал на выходе не имеет жидкой формы, он будет выдан в виде обычного твёрдого вещества
Этот модуль специализирован для параллельной работы
